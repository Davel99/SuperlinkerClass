These are the boring links without Superlinker ;)
Start with the default configuration of Superlinker! It's easy!

"Superlinker" allows your links to be aviable in your parent element, so your visitors will be able to get to your links without searching the small letters :D

You just have to load the script "superlinker.js" in your HTML document, it contains the "superlinker" class

First, add the "superlink-blank" or "superlink" classes to your "a" tag in your HTML, just like this:

<a class="superlink-blank" href="yourURL"> Facebook </a>
<a class="superlink" href="yourURL"> Instagram </a>
» this will open the link in another tab
» this will open the link in the actual tab

Now, let's see what's next!

This is the default configuration for Superlinker!

Once you have prepared your workspace, you just have to add the next javascript code:

var superlink = new superlinker();
superlink.exe();
» this will start the "superlinker" class in your variable
» this will execute the class method which configurates your links
Check the result up there!

The default configuration of superlink applies color "rgba(255,255,255,0.3)" on mouseenter event and makes the font 1.4 times bigger. It also makes your cursor to be a pointer ;)

Now you can click on any place of the parent box of the link! It's comfortable and beautiful :)

What about the Superlinker personalization!?

You can change the background color and fontsize on the hover events!, see the example:

Upps! We need to change the classes for the superlink because they are different from the ones we have up there, just add the "blankClass" and "selfClass" parameters when you initialize the Superlinker class

var superlink = new superlinker({blankClass: 'nc-superlink-blank', selfClass: 'nc-superlink', hover: {color: 'rgba(144, 0, 0, 0.3)', fontResize: 3}});
superlink.exe();
» this will start the "superlinker" with the following hover characteristics: background color will change to rgba(144, 0, 0, 0.3) and the fontResize will increase 3 times :)
» this will execute the method with your configurations ;)

But i just want the link in the parent element!

"Superlinker" adds a class to the parent element, so you can desactivate the hover effects and modify them in your CSS file :)

Follow the example:

var superlink = new superlinker({blankClass: 'lb', selfClass: 'l', hoverClass: 'hovered', hover: {fontResize:false, color:false}});
superlink.exe();
» this will apply "hovered" class name to the parent element, so you can change the hover effects with CSS. It also desactivates the hover configuration that the Superlink class makes
» this will execute the method with your configurations ;)

The "hovered" class is the default class name for the parent element, but you can replace it for whatever you want. If you want to delete ALL hover effects you just have to add "{hover:false}" and it won't be there anymore (including the cursor pointer change), but you can click the parent element and you will still get to your URL :)
